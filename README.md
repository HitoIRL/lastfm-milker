## LastFm Milker
Simple last.fm automatic song scrobbler
so you can milk your favorite song as much as you want to.

## Setup
 1. Rename `Config.toml.example` file to `Config.toml`
 2. Create last.fm application (https://www.last.fm/api/account/create)
 3. Fill `Config.toml` with your app details and songs
 4. Compile and run it lol?

## Limitations
As far as I know last.fm has limit of 3000 scrobbles per day per account
